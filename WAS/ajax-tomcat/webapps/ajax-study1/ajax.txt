AJAX (Asynchronous Javascript And Xml )
: JavaScript와 XML(or JSON)을 이용한 비동기 통신

AJAX를 이용한 통신의 장점 
Page Loading없이 필요한 데이터만 응답받는다. 
-> Client가 Appliaction을 이용할 때 사용자 편의성을 높인다.
-> Page Loading이 없다. 
    새로운 Page로 응답받는 것이 아니라 필요한 데이터만 응답받는다. 
 -> Server의 데이터 전송량 감소 
     
JSON(or XML)을 사용하는 이유
: Ajax 통신시 Server와 Client가 구조화된 데이터를 효과적으로 송수신하기 위해 사용 
 초기에는 XML을 이용했으나 개발의 생산성과 성능을 위해 최근에는 JSON으로 대체되었다.
